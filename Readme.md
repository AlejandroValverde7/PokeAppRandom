## PokeAppRandom

## Indice.
    - • ¿Qué es?
    - • ¿Como funciona?

## 1 - ¿Que es?
    Pequeña aplicacion web que muestra un equipo aleatorio de pokemon's
    para jugar batallas contra tus amigos

## 2 - ¿Como funciona?
    Al pulsar el boton nos muestra un equipo aleatorio por medio de una api externa que nos ofrece
    nombres imagenes y todos los datos de cada pokemon que se muestra

## 3 - Como utilizarlo.
    -Uso de Vite como herramienta de desarollo.
        -Instalar Vite
        -npm i (Instalación)
        -npm run dev (probar en local)

    -Una vez echo esto podemos usarla, si no, podremos utilizarla
    en https://pokeapprandomvx.netlify.app/ donde esta desplegada.

## 4 - Api usada
    -Se utiliza la api https://pokeapi.co/ la cual se puede consultar
    por medio de su propia web o ayudandote con postman

## 5 - Demo desplegada
